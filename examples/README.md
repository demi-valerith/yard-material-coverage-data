# Widget examples

The widget is a standards-based Web Component. Load it once, then use the same
custom element from plain HTML or a framework.

- [`html/index.html`](html/index.html) shows every supported attribute.
- [`react/YardMaterialCoverage.jsx`](react/YardMaterialCoverage.jsx) wraps the
  custom element with React-friendly props.
- [`astro/YardMaterialCoverage.astro`](astro/YardMaterialCoverage.astro) loads
  the component once and renders it from Astro.
- [`wordpress/shortcode.txt`](wordpress/shortcode.txt) shows the separate
  WordPress plugin shortcode.

The npm module performs no analytics, storage, cookies, or background network
requests. When attribution is visible, its source URL contains UTM parameters
that are transmitted only if a visitor clicks the link.
